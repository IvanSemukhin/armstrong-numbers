Задача:
Натуральное десятичное n-значное число называется числом Армстронга,
если сумма его цифр, возведенных в степень n, равна самому числу.
Примеры: 153 = 1^3 + 5^3 + 3^3; 1634 = 1^4 + 6^4 + 3^4 + 4^4.
Найти все числа Армстронга в диапазоде от 0 до 2^63-1

Идея:
Уменьшить класс исследуемых чисел: можно делать перебор не всех чисел, а значений, которые могут получаться в результате степенной суммы.
От перемены цифр местами в числе степенная сумма не меняется.
Т.е. незачем рассматривать все числа из класса : 135, 153, 315, 351, 531 и 513.
Достаточно рассмотреть одно из них, например, число 135.
Вычислить его степенную сумму : (135)ст = 153, а потом убедиться в том что число 153 - это число Армстронга.
Также заранее посчитать степени разрядов и записать их в таблицу.
